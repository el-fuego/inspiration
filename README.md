Graph
===========

Построение графиков в виде линий, столбцовых и круговых дигарамм.<br />
JavaScript, SVG

Пример использования:
<pre>
// Круговая диаграмма
new Graph ({
      el:    'body',
      width:  300,
      height: 300
}).renderCircleDiagram([2, 5, 7, 3, 5]);
</pre>


<pre>
// Линейный график
.renderLine(..);

// Столбцовая диаграмма
.renderDiagram(..);
</pre>
