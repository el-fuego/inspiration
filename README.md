
Colors
===========
Библиотека для работы с цветами в изображении. <br />
JavaScript + Canvas

* Определение основных цветов изображения
* Определение цветов, наиболее близким к заданным
* Определение человекопонятных наименований (н., насыщенный оранжевый)
